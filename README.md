# Weight Converter
A program that converts a weight in pounds and ounces to a weight in kilograms and grams.

## Setup
1. Download and install [Git](https://git-scm.com/downloads), if you don't already have it.

2. Open terminal and run the git clone command:

   ```
   $ git clone https://github.com/cramaechi/weight-converter.git
   ```
    or [download as ZIP](https://github.com/cramaechi/weight-converter/archive/master.zip).

3. Go to the project folder:

   ```
   $ cd weight-converter
   ```

4. Compile:

   ```
   $ make
   ```
   
## Usage
Run:

```
$ ./weight_converter
```
